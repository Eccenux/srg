## Prosty serwer

Instalacja prostego serwera (do testów ładowania plików):
```bash
npm install -g http-server
```

 Uruchomienie na wybranym porcie:
```bash
http-server . -p 8087
```
(uruchomienie nie działa w `powershell`, ale `cmd` jest OK)
